News

March 2nd, 2009 - The first official alpha release of Kantan SGF - kantan-sgf_0.0.1.

Overview

Kantan SGF is a Ruby Simple Game Format parser, written using Treetop's Parser Expression Grammars. It fills a niche in the Ruby world as a standalone sgf parser.

Release

Current code releases can be found at the Rubyforge project page.

Source

The source code can be found at the github project page.

Gem Install

	$ gem install treetop
	$ gem install kantan-sgf
